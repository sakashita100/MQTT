- 腕
  - Pythonの実行ボタンを押したら、腕が動くようになる。
  - 1回押すと、自動で戻る。
  - モバイルバッテリー(短いほう)と電源(長いほう)を入れてから、腕を動かす。
  - Pythonプログラムの内容
  - 文字数が4文字だったら腕が動く
  - 文字数が3文字だったら腕が動かない
  - 毎回押さないと動かないので注意する
  - wifiの書き換えが必要
